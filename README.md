# Archaeo-LLM

Studentská webová prezentace projektu Archaeo-LLM zaměřeného na databázovou evidenci archeologických nálezů se zaměřením na byzantské mince.

## Obsah projektu

- jednoduchá webová prezentace v HTML a CSS
- databázová dokumentace
- galerie vybraných byzantských mincí
- kontaktní formulář
- veřejný zdroj dat: Portable Antiquities Scheme (PAS)

## Struktura webu

- index.html
- pages/database.html
- pages/gallery.html
- pages/about.html
- pages/contact.html

## Použité technologie

- HTML
- CSS
- Visual Studio Code
- GitHub Pages
## Zdroj dat a obrázků

Portable Antiquities Scheme:
https://finds.org.uk/database/search/results/q/COIN/broadperiod/BYZANTINE
